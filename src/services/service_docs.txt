Louer Backend API

Link to docs: https://docs.google.com/document/d/1aeQMNXxsgrZa31rFysfn2E3RGnqQP_Tc0jB64aSvU7c/

1. JSONs:

User JSON:
+ userId: Long
+ studentId: String
+ firstName: String
+ lastName: String
+ surname: String
+ email: String
+ phone: String
+ rating: Float (0 - 100)
+ userStatus: boolean (1:ok, 0:ban)
+ DoB: NGÀY sinh
+ gender: boolean



Product JSON:
+ productId: Long
+ productName: String
+ brand: JSON:
---brandId: Long
---brandName: String
---numberOfProducts: Integer
+ category: JSON:
---categoryId: Long
---categoryName: String
---numberOfProducts: Integer
+ numberOfProducts: Integer
+ marketPrice: Integer



Listing JSON:
+ listingId: Long
+ product: Product JSON
+ user: User JSON
+ listingDescription: String  
+ price: Integer
+ listingStatus: boolean



Order JSON:
+ orderId: Long
+ listing: Listing JSON
+ lessee: User JSON
+ lessor: User JSON
+ basePrice: Integer
+ duration: Integer
+ price: Integer
+ dateStart: NGÀY và GIỜ
+ dateEnd: NGÀY và GIỜ
+ orderStatus: Integer
0: pending
1: in progress
2: completed
3: canceled



Image JSON:
+ imgId: Long
+ listingId hoặc productId hoặc userId hoặc messageId: Long
+ imgLink: String, link ảnh







2. Requests:

# Profile 1 user:
- Gửi GET request đến: (url)/users/{usersId}
=> User JSON

# Lấy hình ảnh cho users:
- Gửi GET request đến: (url)/images/users?user_id=

# Thêm hình ảnh cho users:
- Gửi POST request đến: (url)/images/users?user_id=

# Xóa hình ảnh của users:
- Gửi DELETE request đến: (url)/images/users?user_id=

# Update profile 1 user:
- Gửi PUT request đến: (url)/users/{userId}

Gửi trong request body:
+ firstName: String
+ lastName: String
+ surname: String
+ phone: String
+ DoB: NGÀY sinh
+ gender: boolean




# Tất cả products trong 1 page:
- Gửi GET request đến:  (url)/products?page=n&size=n
=> Product JSONs

# Lấy hình ảnh cho products:
- Gửi GET request đến: (url)/images/products?product_id=


# Tất cả listings của 1 product:
- Gửi GET request đến: (url)/listings?productId=
# Tất cả listings của 1 user:
- Gửi GET request đến: (url)/users/{userId}/listings
# Chi tiết 1 listing:
- Gửi GET request đến: (url)/listings/{listingId}
=> Listing JSONs

# Lấy hình ảnh cho listings:
- Gửi GET request đến: (url)/images/listings?listing_id=

# Thêm listing:
- Gửi POST request đến: (url)/listings

Gửi về trong request body:
+ productName: Long
+ brandName: String
+ categoryName: String
+ listingDescription: String
+ marketPrice: Integer, giá thị trường của sản phẩm
+ price: Integer, giá cho thuê của sản phẩm

# Thêm hình ảnh cho listings:
- Gửi POST request đến: (url)/images/listings?listing_id=
# Thêm hình ảnh cho products:
- Gửi POST request đến: (url)/images/products?product_id=
(Hình ảnh của listings dùng làm cả hình ảnh products, khi user upload ảnh listing, gửi luôn về url product)

# Update listing:
- Gửi PUT request đến: (url)/listings?listingId=

Gửi về trong request body:
+ productName: Long
+ brandName: String
+ categoryName: String
+ listingDescription: String
+ marketPrice: Integer, giá thị trường của sản phẩm
+ price: Integer, giá cho thuê của sản phẩm

# Gỡ listing:
- Gửi DELETE request đến: (url)/listings/remove?listingId=


# Tất cả order trên 1 page, order đến từ người khác (lessor):
- Gửi GET request đến: (url)/users/{userId}/lessor/orders?page=n&size=n
# Tất cả order trên 1 page, order tới người khác (lessee):
- Gửi GET request đến: (url)/users/{userId}/lessee/orders?page=n&size=n
# Tất cả order trên 1 page, order chung:
- Gửi GET request đến: (url)/users/{userId}/orders?page=n&size=n

# Thông tin 1 order (lessor):
- Gửi GET request đến: (url)/users/{userId}/lessor/orders?orderId=
# Thông tin 1 order (lessee):
- Gửi GET request đến: (url)/users/{userId}/lessee/orders?orderId=
# Thông tin 1 order (chung):
- Gửi GET request đến: (url)/users/{userId}/orders?orderId=
# Thanh toán 1 order:
-Gửi GET request đến: (url)/users/{userId}/lessee/orders/checkout?orderId=
=> Order JSONs

# Thêm request xin thuê:
- Gửi POST request đến:  (url)/request?listingId=

Gửi về trong request body:
+ senderId: Long, id người đi thuê
+ receiverId: Long, id người cho thuê
+ lease_duration: Integer, số giờ thuê

#Xem requests xin thuê sản phẩm:
- Gửi GET request đến:  (url)/request?listingId=

Gửi trong request body:
+ requestId: Long
+ sender: user json
+ receiver: user json
+ listing: listing json
+ leaseDuration: Integer, số giờ thuê
+ requestStatus: Integer

# Mở order:
- Gửi POST request đến: (url)/listings/{listingId}/orders

Gửi trong request body:
+ lesseeId: Long
+ lessorId: Long
+ duration: Integer, số giờ thuê
+ dateStart: ngày và giờ dự kiến bắt đầu thuê
+ dateEnd: ngày và giờ dự kiến kết thúc

# Update order:
- Gửi PUT request đến: (url)/orders?orderId=

Gửi trong request body:
+ duration: Integer, số giờ thuê
+ dateStart: ngày và giờ dự kiến bắt đầu thuê
+ dateEnd: ngày và giờ dự kiến kết thúc

# Cancel order:
- Gửi PUT request đến: (url)/orders/cancel?orderId=
